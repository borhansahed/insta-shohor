Javascript single threaded, synchronised language. This means that when our code is run in the browser, the browser engine reads the code from the top of the file at once and then goes downwards.Not going to the next line until the first line is complete.

Basically javascript can read first code and then go another. Asynchronous function has created to solve this problem.The functions that are there will be sent from the serial wise callback queue to the call stack. And it loops all the tasks and sends them to the call stack, that's why it's basically called an event loop.
